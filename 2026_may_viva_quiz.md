### HTML, CSS & JavaScript Fresher Quiz (30 Minutes)

#### Instructions:
- Read each question carefully.
- Choose the best possible answer from the options provided.
- There are three sections: HTML (10), CSS (10), and JavaScript (20).

---

### Part 1: HTML (10 Questions)

**1. What does the acronym HTML stand for?**
A) Hyper Transfer Markup Language
B) Hyper Text Markup Language
C) High Tech Modern Language
D) Hyperlink and Text Markup Language

**2. Which HTML tag is used to define the main content of a document, which should be unique to that document?**
A) `<content>`
B) `<body>`
C) `<main>`
D) `<article>`

**3. What is the correct HTML for creating a hyperlink?**
A) `<a url="http://www.example.com">Example</a>`
B) `<link href="http://www.example.com">Example</link>`
C) `<href="http://www.example.com">Example</a>`
D) `<a href="http://www.example.com">Example</a>`

**4. Which tag is used to create an ordered list?**
A) `<ul>`
B) `<ol>`
C) `<list>`
D) `<dl>`

**5. What is the purpose of the `alt` attribute in an `<img>` tag?**
A) It specifies the alignment of the image.
B) It provides alternative text for an image if it cannot be displayed.
C) It sets the URL of the image.
D) It defines the title of the image.

**6. Which of the following is NOT a semantic HTML5 element?**
A) `<article>`
B) `<header>`
C) `<div>`
D) `<nav>`

**7. Which input type defines a field for entering a password?**
A) `<input type="text">`
B) `<input type="hidden">`
C) `<input type="password">`
D) `<input type="secure">`

**8. How do you add a comment in an HTML file?**
A) `// This is a comment`
B) `/* This is a comment */`
C) `<!-- This is a comment -->`
D) `# This is a comment`

**9. Which element is considered a block-level element by default?**
A) `<span>`
B) `<a>`
C) `<img>`
D) `<p>`

**10. What is the correct way to specify the character encoding for an HTML document?**
A) `<meta encoding="UTF-8">`
B) `<meta charset="UTF-8">`
C) `<charset type="UTF-8">`
D) `<head charset="UTF-8">`

### Part 2: CSS (10 Questions)

**11. What does CSS stand for?**
A) Creative Style Sheets
B) Computer Style Sheets
C) Cascading Style Sheets
D) Colorful Style Sheets

**12. Which is the correct syntax to link an external stylesheet named `style.css`?**
A) `<style src="style.css">`
B) `<stylesheet>style.css</stylesheet>`
C) `<link rel="stylesheet" type="text/css" href="style.css">`
D) `<link rel="stylesheet" href="style.css" type="css/text">`

**13. How would you select an element with the id `header` in CSS?**
A) `.header`
B) `*header`
C) `header`
D) `#header`

**14. Which property is used to change the background color of an element?**
A) `color`
B) `bgcolor`
C) `background-color`
D) `background`

**15. In the CSS box model, which property creates space *around* an element, outside of its border?**
A) `padding`
B) `margin`
C) `border`
D) `spacing`

**16. Which CSS property controls the text size?**
A) `font-style`
B) `text-size`
C) `font-size`
D) `text-style`

**17. What does the `display: none;` property do?**
A) It makes the element transparent.
B) It hides the element and removes it from the document layout.
C) It moves the element to the back.
D) It hides the element but keeps the space it occupied.

**18. Which pseudo-class is used to style a link when the user's mouse is over it?**
A) `:hover`
B) `:active`
C) `:focus`
D) `:visited`

**19. Which of these selectors has the highest specificity?**
A) `div p`
B) `.content`
C) `#main-title`
D) `h1`

**20. How do you group multiple selectors to apply the same style?**
A) Separate each selector with a space.
B) Separate each selector with a plus sign (+).
C) Separate each selector with a comma (,).
D) Separate each selector with a greater-than sign (>).

### Part 3: JavaScript (20 Questions)

**21. Where is the correct place to insert a JavaScript file link in HTML?**
A) In the `<head>` section.
B) In the `<body>` section.
C) Both the `<head>` and `<body>` sections are correct.
D) In the `<footer>` section.

**22. How do you create a variable that can be reassigned in modern JavaScript?**
A) `var myVar = 5;`
B) `let myVar = 5;`
C) `const myVar = 5;`
D) `variable myVar = 5;`

**23. What will the following code output to the console? `console.log(typeof "Hello");`**
A) `string`
B) `text`
C) `String`
D) `object`

**24. Which operator is used for strict equality (checks both value and type)?**
A) `==`
B) `=`
C) `===`
D) `!=`

**25. How do you write a single-line comment in JavaScript?**
A) `<!-- This is a comment -->`
B) `/* This is a comment */`
C) `// This is a comment`
D) `# This is a comment`

**26. What is the correct way to select an HTML element with the id `demo` in JavaScript?**
A) `document.getElement("demo")`
B) `document.querySelector("#demo")`
C) `document.getElementByName("demo")`
D) `document.select("#demo")`

**27. How do you define a function in JavaScript?**
A) `function: myFunction() {}`
B) `function myFunction() {}`
C) `define myFunction() {}`
D) `myFunction function() {}`

**28. Which event occurs when the user clicks on an HTML element?**
A) `onmouseclick`
B) `onmouseover`
C) `onclick`
D) `onchange`

**29. How do you add an element `apple` to the end of an array called `fruits`?**
A) `fruits.add("apple");`
B) `fruits.push("apple");`
C) `fruits.append("apple");`
D) `fruits[fruits.length] = "apple";`

**30. What will be the result of the following comparison: `'5' == 5`?**
A) `true`
B) `false`
C) `undefined`
D) `TypeError`

**31. What is the purpose of the `return` statement in a function?**
A) It stops the execution of the function.
B) It returns a value from the function.
C) It prints a value to the console.
D) Both A and B are correct.

**32. Which loop is guaranteed to execute at least once?**
A) `for` loop
B) `while` loop
C) `do...while` loop
D) `for...in` loop

**33. How do you create an object literal in JavaScript?**
A) `var person = ["John", "Doe"];`
B) `var person = {"firstName": "John", "lastName": "Doe"};`
C) `var person = ("John", "Doe");`
D) `var person = new Object("John", "Doe");`

**34. What does `NaN` stand for?**
A) Not a Number
B) No and No
C) New-age Number
D) Not a Name

**35. What is the correct way to get the first element from an array named `myArray`?**
A) `myArray.first()`
B) `myArray[1]`
C) `myArray[0]`
D) `myArray.get(0)`

**36. Which method converts a JSON string into a JavaScript object?**
A) `JSON.stringify()`
B) `JSON.parse()`
C) `JSON.objectify()`
D) `JSON.toObject()`

**37. Which of the following is NOT a primitive data type in JavaScript?**
A) `string`
B) `number`
C) `boolean`
D) `array`

**38. What is the purpose of `addEventListener()`?**
A) To create a new event.
B) To remove an event from an element.
C) To attach an event handler to a specified element.
D) To trigger an event programmatically.

**39. What will `console.log(3 + 2 + "7")` output?**
A) `"12"`
B) `"327"`
C) `"57"`
D) `12`

**40. A variable declared with `const` must be...**
A) A number.
B) Assigned a value when it is declared and cannot be reassigned.
C) Global in scope.
D) Reassigned later in the code.
